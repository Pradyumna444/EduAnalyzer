Project Report: Class Performance Heatmap and Teaching Style Analyzer1. AbstractThe "Class Performance Heatmap and Teaching Style Analyzer" is a web-based analytics platform designed to bridge the gap between student academic outcomes and pedagogical methods. Traditional academic grading systems fail to provide immediate visual cues regarding class-wide trends. This project solves this by generating an interactive performance heatmap. Furthermore, it incorporates a feedback loop that evaluates faculty teaching styles (Clarity, Interaction, Support) and correlates them with student performance, providing automated suggestions for academic improvement.2. IntroductionIn modern education, data analytics plays a vital role in shaping teaching methodologies. This project serves as a dual-purpose dashboard. For students, it identifies areas of weakness through color-coded matrices (Heatmaps) and correlates attendance with grades. For teachers, it aggregates anonymous student feedback to categorize their teaching style (e.g., "Interactive & Engaging", "Lecture-heavy") and offers actionable recommendations.2.1 ObjectivesTo create a visual mapping of student scores across subjects using Heatmaps.To automatically flag 'At-Risk' (weak) students for early intervention.To analyze teacher feedback metrics and classify teaching styles.To provide a unified, user-friendly interface for educational administrators.3. 4.3 Modules DescriptionData Collection Module: Handles insertion of marks, attendance, and qualitative/quantitative teacher feedback.Heatmap Visualization Module: Transforms raw scores into a normalized matrix, applying conditional CSS classes (Red, Yellow, Green) for immediate visual assessment.Student Analysis Module: Calculates aggregate averages, flags students falling below a threshold, and maps attendance-to-performance correlations using Scatter plots.Teaching Style Analyzer: Uses a rules-based algorithm on feedback metrics to deduce pedagogical style and generates automated text-based suggestions.5.
